点击身份的ID，可以直接查看身份的详细信息，其中包含了钱包、资源、以及和脚本交互的记录。以及这个身份的虚拟姓名、性别、地区、头像等社交信息，这些都是由AI生成的，在一些社交媒体相关的任务中，任务总是会使用AI设定的社交身份来表明自己的信息。
